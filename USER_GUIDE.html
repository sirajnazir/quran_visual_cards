1. Project Overview & Architecture

The Quran Visual Cards Pipeline is a full-stack application that analyzes Quranic verses through the lens of mental models and generates beautiful hand-drawn sketchnote-style visual cards.

System Components

ComponentTechnologyPurpose
Backend APIFlask (Python)REST endpoints, orchestration
Text AnalysisClaude Sonnet (Anthropic)Verse analysis, model matching, back-card text
Image GenerationGemini (Google AI)Hand-drawn card illustrations
Model RepositoryJSON database103 mental models, 15 categories, 20 visual types
Matching EnginePython (weighted scoring)Intelligent model-to-verse matching
Islamic GuardrailsPython (prompt injection)Ensures Islamic art compliance
FrontendHTML + Rough.jsWeb UI, preview, gallery

Data Flow

Input Verse/Analysis
    ↓
[intelligent_matching.py] → Pre-score against 103 models
    ↓
[core_intelligence.py]  → Claude analyzes + selects best model & visual type
    ↓
[islamic_guardrails.py] → Detect Islamic content, inject guardrails
    ↓
[image_gen.py]          → Build prompt (20 visual types) + generate via Gemini
    ↓
Output: Card image (PNG) + structured card data (JSON)

2. Setup & Installation

Prerequisites

  • Python 3.8+
  • pip (Python package manager)
  • Anthropic API key (for text analysis via Claude)
  • Google AI API key (for image generation via Gemini) — optional, can use NanoBanana MCP instead

1 Install Dependencies

# Standard install
pip install flask anthropic google-genai Pillow

# macOS / system Python (if you get permission errors)
pip install flask anthropic google-genai Pillow --break-system-packages

2 Set API Keys

# Option A: Environment variables (recommended)
export ANTHROPIC_API_KEY="sk-ant-api03-..."
export GOOGLE_AI_API_KEY="AIzaSy..."

# Option B: Add to your shell profile (~/.zshrc or ~/.bash_profile)
echo 'export ANTHROPIC_API_KEY="sk-ant-api03-..."' >> ~/.zshrc
echo 'export GOOGLE_AI_API_KEY="AIzaSy..."' >> ~/.zshrc
source ~/.zshrc
Note: The Anthropic key is required for verse analysis (Claude). The Google key is needed only for direct Gemini API image generation. If you use NanoBanana MCP for images (see Section 9), you only need the Anthropic key.

3 Copy Existing Generated Images (Optional)

# If you previously generated images via NanoBanana MCP
sh setup_images.sh
Expected output: [OK] Copied 5 of 5 images.

4. API: Analyze Verse (Core Intelligence)

The analysis endpoint is the heart of the system. It takes a Quranic verse title, category, and analysis text, then returns a fully structured card with the best-matching mental model, visual type, steps, Islamic content flags, and more.

POST /api/analyze
Analyze input text using Claude + intelligent matching engine

Test 1: Islamic Content (Quranic Verse)

curl -X POST http://localhost:5000/api/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "title": "The Tawakkul Chain",
    "category": "Faith & Action",
    "analysis_text": "Surah At-Talaq 65:3 teaches that whoever places trust in Allah, He is sufficient for them. Tawakkul is not passive resignation but an active tripod: preparation (taking means), delegation (entrusting outcomes to Allah), and acceptance (contentment with results). Without preparation, trust becomes negligence. Without delegation, effort becomes arrogance. Without acceptance, devotion becomes anxiety."
  }'

Expected Response Fields

FieldTypeExample
model_typestring"tripod"
visual_typestring"tripod"
model_idstring"tawakkul_trust_in_allah"
matched_model_namestring"Tawakkul (Trust in Allah)"
model_confidencefloat0.85
themestring"warm_parchment"
is_islamic_contentbooltrue
islamic_elementsarray["geometric_patterns", "crescent_star"]
stepsarray3 step objects with label, description, arabic

Test 2: Non-Islamic Content (General Mental Model)

curl -X POST http://localhost:5000/api/analyze \
  -H "Content-Type: application/json" \
  -d '{
    "title": "The Eisenhower Matrix",
    "category": "Productivity",
    "analysis_text": "President Eisenhower organized tasks into four quadrants based on urgency and importance. Urgent-Important tasks are done immediately. Important but Not Urgent tasks are scheduled. Urgent but Not Important tasks are delegated. Neither Urgent nor Important tasks are eliminated. Most people spend too much time in the Urgent quadrants, neglecting the Important-Not-Urgent quadrant where real growth happens."

13. Troubleshooting

Common Issues

"No module named 'flask'" (or anthropic, google.genai, PIL)

pip install flask anthropic google-genai Pillow --break-system-packages

"No Google API key configured"

# Set the environment variable
export GOOGLE_AI_API_KEY="AIzaSy..."

# Or pass it in the request body
curl -X POST ... -d '{"google_api_key": "AIzaSy...", ...}'

"Analysis failed" / Empty analysis response

# Check your Anthropic API key
export ANTHROPIC_API_KEY="sk-ant-api03-..."

# Test the key directly
python3 -c "
import anthropic
c = anthropic.Anthropic()
r = c.messages.create(model='claude-sonnet-4-20250514', max_tokens=100,
    messages=[{'role':'user','content':'Say hello'}])
print(r.content[0].text)
"

Images not showing in gallery.html / preview.html

# DON'T open as file:// — use HTTP server instead
python3 -m http.server 8080
# Then open http://localhost:8080/gallery.html

setup_images.sh "declare -A" error on macOS

# Use sh instead of bash (the script is POSIX-compatible)
sh setup_images.sh

Image generation quota exceeded

Google Gemini free tier has rate limits. Options:

  • Wait and retry (limits reset every minute)
  • Use /api/generate-prompt and paste into NanoBanana MCP
  • Upgrade to a paid Google AI plan
